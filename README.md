## Features

### Snake Game

The portfolio includes an interactive Snake game built with Phaser.js, showcasing game development skills:

- **Controls**: Arrow keys or WASD for movement
- **Pause/Resume**: Press 'P' to pause/unpause the game
- **Progressive Difficulty**: Game speed increases every 50 points
- **Score System**: Earn 10 points for each food item consumed
- **Restart**: Press SPACE to restart after game over
- **Responsive Design**: Fits within the portfolio's design system

The game is located on the home page and demonstrates proficiency in:

- Game development with Phaser.js
- TypeScript integration
- React component architecture
- State management
- Event handling
